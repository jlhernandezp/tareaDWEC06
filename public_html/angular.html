AngularJS es un framework MVC (modelo, vista, controlador)de JavaScript para el Desarrollo Web Front End que permite crear aplicaciones SPA (Single-Page Applications)

En una interfaz SPA todas las vistas de la interfaz de la aplicación están contenidas en la SPA, en el cliente, realizando una única carga inicial y posponiendo recursos pesados: Grandes cantidades de datos, Imágenes, videos, …

AngularJS tiene entre otros muchos objetivos crear aplicaciones SPA y a su vez mantener las ventajas del sistema de carga de URLs.

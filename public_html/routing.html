Es un módulo de Angularjs permite la navegación por diferentes páginas sin necesidad de recargarla ( aplicación SPA)

Para su correcta utilización debemos tener un contenedor, capa div en el que se introducirá el contenido a mostrar, está definido por la directiva ng-View. con el servico $routeProvider le indicamos que páginas o código html vamos a introducir en el contenedor ng-View.

A estas páginas o plantillas o vistas se les pueden aplicar elementos controller que, utilizando expresiones del framework Angularjs permiten interactuar con ellas
